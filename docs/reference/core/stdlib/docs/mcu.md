# MCU

This module enables the access to low level microcontroller functionalities like:


* unique identifiers


* soft reset


---
#### `#!py3 reset()`

!!!abstract "`#!py3 reset()`"

Restart the microcontroller.


---
#### `#!py3 uid()`

!!!abstract "`#!py3 uid()`"

Return a bytes object containing the unique id of the mcu
